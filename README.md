# PalindroneCheckerApp

## Objective

This project checks whether a word or number is a palindrome using Java.

## What is a Palindrome

A palindrome is a word or number that reads the same forward and backward.
Example: madam, level, 121

## Technologies Used

Java

## Running Procedure

1. Compile the program: `javac PalindromeCheckerApp.java`
2. Run the program: `java PalindromeCheckerApp`

## Flow

User Input → Check Palindrome → Display Result
